# Construction Terminology Reference

Spanish-to-English construction terms commonly found in plan sets by Central American and Caribbean architects. Organized by construction phase.

## Foundation

| Spanish | English | Description |
|---------|---------|-------------|
| Zapata | Pad footing | Isolated concrete pad under a column |
| Zapata tipo 1 / tipo 2 | Footing type 1 / type 2 | Different sizes for different load conditions |
| Cimiento corrido | Continuous strip footing | Runs under walls to distribute load |
| Solera de humedad | Moisture barrier beam | RC beam at grade level preventing ground moisture wicking |
| Solado | Blinding layer | Thin concrete leveling course poured over soil before foundation work |
| Sobrecimiento | Stem wall | Masonry or concrete between footing top and floor level |
| Muro de piedra | Stone masonry wall | Rubble stone wall between footing and grade beam |
| Hormigon ciclopeo | Cyclopean concrete | Large stones embedded in concrete matrix (used for sub-grade walls) |
| Arranques | Starter bars | Rebar stubs extending up from foundation for future column connection |
| Recubrimiento | Cover / clear cover | Distance from rebar to nearest concrete surface (protects rebar from corrosion) |

## Walls and Structure

| Spanish | English | Description |
|---------|---------|-------------|
| Columnas de confinamiento (C1-C5) | Tie columns / confinement columns | RC columns poured after masonry walls, providing seismic confinement |
| Eslabones | Stirrups | U-shaped rebar ties wrapping longitudinal bars in columns and beams |
| Dentado | Toothing | Zigzag pattern at masonry wall edges for column-to-wall mechanical interlock |
| Solera intermedia | Mid-height bond beam | Horizontal RC beam at wall mid-height for lateral restraint |
| Solera de corona | Crown bond beam / ring beam | RC beam at top of walls, under roof, tying everything together |
| Bloque de 40x20x15 | Standard CMU | Concrete masonry unit, 40cm x 20cm x 15cm |
| Bloque de 40x20x10 | Thin CMU | Thinner block for non-structural partitions |
| Acero de grado 60 / 70 | Grade 60 / 70 rebar | Steel yield strength in ksi (60,000 or 70,000 psi) |
| Concreto de 4000 PSI | 4,000 PSI concrete | Target compressive strength at 28 days |

## Roof

| Spanish | English | Description |
|---------|---------|-------------|
| Lamina troquelada | Corrugated metal roofing | Typically Cal. 26 gauge galvanized steel |
| Capote | Ridge cap | Metal cover piece at the roof peak |
| Costanera | Purlin | Horizontal roof framing member supporting roofing sheets |
| Tubo galvanizado | Galvanized steel tube | Used for trusses and purlins (e.g., 4"x2", 3"x3") |
| Cielo falso prefabricado | Prefabricated drop ceiling | Suspended ceiling panels below roof structure |
| Viga canal | Gutter beam | Built-in concrete roof gutter (common in Caribbean construction) |
| Tornillo para lamina | Roofing screw | Self-tapping screw with neoprene washer for metal roofing |
| Porta costanera | Purlin bracket | Metal bracket connecting purlin to truss/beam |
| Perno | Bolt | Structural bolt for connections |
| Arandela de presion | Lock washer | Spring washer preventing bolt loosening |
| Roldana | Flat washer | Load-distributing washer |
| Pendiente | Slope / pitch | Roof slope direction and angle |

## Plumbing and Drainage

| Spanish | English | Description |
|---------|---------|-------------|
| Caja de registro | Inspection box / cleanout | Access box for drain maintenance |
| Caja sifon | Siphon trap box | P-trap access box preventing sewer gas backflow |
| Trampa de grasa | Grease trap | Kitchen waste interceptor |
| Aguas negras | Sewage / blackwater | Toilet and contaminated wastewater |
| Agua pluvial | Stormwater / rainwater | Roof and surface runoff |
| Bajada de agua pluvial (BAP) | Rainwater downspout | Vertical pipe carrying roof water to ground drainage |
| Tubo de PVC | PVC pipe | Plastic pipe (sizes in inches: 1/2", 3/4", 1", 2", 3", 4") |
| Codo 90 / 45 | 90 / 45 degree elbow | Pipe direction change fitting |
| Tee horizontal / vertical | Horizontal / vertical tee | T-shaped pipe junction |
| Yee horizontal | Horizontal wye | Y-shaped pipe junction for smooth flow |
| Reductor | Reducer fitting | Transitions between pipe diameters |
| Union de expansion | Expansion joint | Allows thermal expansion/contraction in pipe runs |
| Sifon terminal horizontal | Terminal siphon | End-of-line trap vent |
| Pendiente de tuberias (PEND. 2%) | Pipe slope (2%) | Minimum gravity drainage slope |
| Reposaderas existentes | Existing floor drains | Pre-existing drain points |
| Campo de absorcion | Absorption field / leach field | Subsurface wastewater dispersal area |
| Aguas servidas | Greywater / used water | Non-toilet wastewater (sinks, showers) |
| Drenaje pluvial | Stormwater drainage | Rainwater collection and disposal system |
| Drenaje de lavamanos | Sink drain | Lavatory waste pipe |
| Drenaje de retrete | Toilet drain | Water closet waste pipe |
| Viene de red principal | From main supply | Connection point from municipal or well water |
| N.P.T. (Nivel de Piso Terminado) | F.F.L. (Finished Floor Level) | Reference elevation for finished floor |
| Ladrillo tayuyo | Clay brick | Used for inspection/siphon box construction |
| Alisado de cemento | Cement morite bed | Smooth interior finish of drain boxes |
| Tapa / tapadera de concreto | Concrete cover / lid | Removable access cover for inspection boxes |
| Cortina | Baffle wall | Internal divider in grease trap |
| Camara de sedimentacion | Settling chamber | First compartment of grease trap |
| Camara de agua combinada | Combined water chamber | Second compartment after grease separation |
| Colador de agua | Water strainer | Filter screen at grease trap outlet |

## Electrical

| Spanish | English | Description |
|---------|---------|-------------|
| Contador electrico | Electric meter | Utility metering point (typically at street) |
| Tablero de flipones | Breaker panel / load center | Main distribution panel with circuit breakers |
| Tomacorriente | Electrical outlet / receptacle | Wall-mounted power outlet |
| Tomacorriente doble | Duplex outlet | Double receptacle |
| Interruptor simple | Single-pole switch | Controls one light/circuit |
| Interruptor doble | Double-pole switch | Controls two circuits from one location |
| Interruptor triple | Triple switch | Controls three circuits from one location |
| Lampara / luminaria de cielo | Ceiling light fixture | Overhead lighting |
| Lampara de pared | Wall sconce | Wall-mounted light |
| Lampara reflector | Floodlight | Exterior security/area lighting |
| Conductor positivo forrado | Insulated hot conductor | Live wire (typically black or red) |
| Conductor neutro forrado | Insulated neutral conductor | Return wire (typically white) |
| Conductor retorno forrado | Insulated return conductor | Switch leg wire |
| Conductor puente forrado | Insulated jumper conductor | Bridge/jumper wire between devices |
| Tuberia: poliducto o PVC en cielo | Conduit: flexible or PVC in ceiling | Wire pathway in ceiling space |
| Poliducto de 3/4" | 3/4" flexible conduit | Standard residential conduit size |
| Tierra fisica | Ground / earth | Safety grounding conductor |
| Linea activa | Hot / live line | Current-carrying conductor in conduit |
| Linea neutra | Neutral line | Return conductor in conduit |
| Retorno para timbre | Doorbell return | Low-voltage return wire for doorbell |
| Circuito y unidad No. indicado | Circuit and unit number as indicated | Circuit identification marking |

## Finishes

| Spanish | English | Description |
|---------|---------|-------------|
| Repello + cerinado + pintura (R+C+P) | Plaster + skim coat + paint | Standard 3-layer wall finish |
| Azulejo (AZ) | Ceramic tile | Wall tile (typically in wet areas) |
| Piso ceramico (PC) | Ceramic floor tile | Standard floor finish |
| Piso ceramico antideslizante (PC-AD) | Non-slip ceramic tile | Slip-resistant tile for wet areas (typical size 0.28 x 0.28m) |
| Cielo falso prefabricado (CFP) | Prefabricated drop ceiling | Suspended ceiling panels |
| Torta de concreto (TC) | Concrete slab floor | Poured concrete floor (for porches, utility areas) |
| Engravillado (E) | Gravel surface | Loose gravel ground cover |
| Adoquin (AD) | Paving stone / paver | Interlocking concrete pavers (typical 30 x 25 x 10 cm) |
| Dintel | Lintel | Reinforced beam above door/window opening |
| Sillar | Sill | Bottom of window opening (height from floor) |
| Ancho vano | Opening width | Clear width of door/window rough opening |
