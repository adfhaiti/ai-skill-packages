# Functional Expense Allocation Guide

Detailed methodology for allocating expenses across the three functional categories required by Form 990 Part IX: Program Services, Management and General, and Fundraising.

## Core Principles

### The Three Functions

| Function | Definition | Examples |
|---|---|---|
| **Program Services** | Activities that fulfill the organization's exempt purpose | Direct service delivery, grants to beneficiaries, program staff salaries, program supplies, field operations, training participants |
| **Management and General (M&G)** | Activities that support the organization but are not directly identifiable with program or fundraising | Executive management, finance/accounting, HR, board governance, legal compliance, general insurance, audit fees, overall planning |
| **Fundraising** | Activities undertaken to solicit contributions, grants, or other forms of financial support | Grant writing, donor cultivation, fundraising events, direct mail campaigns, donor database management, gift processing, donor acknowledgment |

### Allocation Hierarchy

Apply in this order for each expense:

1. **Direct assignment**: if the expense serves only one function, assign 100% to that function
2. **Specific identification**: if the expense serves multiple functions but can be traced to specific activities, allocate based on the activity split
3. **Reasonable allocation base**: if the expense is shared and cannot be traced, allocate using a documented, reasonable basis

### Reasonable Allocation Bases

| Allocation Base | Best Used For | How to Calculate |
|---|---|---|
| Time/effort | Salaries, benefits, payroll taxes for staff splitting time across functions | Time studies, job descriptions, timesheet analysis; % of time on program vs M&G vs fundraising |
| Square footage | Occupancy, utilities, maintenance, depreciation on buildings | Measure space dedicated to each function; shared spaces allocated by headcount or usage |
| Headcount | Office supplies, IT infrastructure, telephone, general insurance | Count FTEs (or proportion of FTEs) assigned to each function |
| Direct cost ratio | Overhead items that correlate with spending volume | Ratio of direct costs by function (e.g., if program = 75% of direct costs, allocate 75% of overhead to program) |
| Usage/consumption | Printing, postage, vehicle expenses, equipment depreciation | Track actual usage by function where feasible |
| Transaction count | Accounting, banking fees | Count transactions by function |

### Documentation Requirements

For every allocation:
- State the allocation base used
- Explain why it is reasonable for that expense type
- Document the data source (time study date, square footage measurement, headcount as of date)
- Apply consistently year over year (changes in method require disclosure)

Organizations must maintain the following documentation systems (per Blazek & Adams):

**1. Staff Salary Allocation System (Essential)**
Each staff member should maintain an individual time record (computer database, timesheet, or time-tracking app). Reports should be completed weekly for accuracy. For personnel performing repetitive tasks, preparing one week's report per month or one month per year may suffice. Percentages of time spent on various functions are then tabulated and used for accounting allocations. This is the single most important documentation for functional expense allocation.

**2. Office/Program Space Utilization Charts**
All physical building space (rented or owned) must be allocated according to usage. Prepare floor plans tabulated by square footage allocable to each activity center. For dual-use space, maintain records of the number of hours or days the space is used for each purpose.

**3. Direct Program Cost Capture**
Establish separate vendor accounts, cost codes, or department codes for different programs. Benefits include: reduction of unrelated business income, proof of qualifying distributions (if applicable), and defense against IRS challenges of low program expenditure ratios. Examples: coded long-distance telephone charges by department, separate supply accounts by program.

**4. Joint Project Allocation Records**
Four recognized allocation methods:
- Activity-based: identify departmental costs directly
- Equal sharing: divide equally among benefiting functions (e.g., 3 programs = 1/3 each)
- Stand-alone cost: what each component would cost if performed independently
- Cost-savings proportional: allocate in proportion to savings realized by sharing

**5. Computer-Based Fund Accounting System**
A system where department/function codes are automatically recorded as expenditures are made is strongly preferred. QuickBooks Classes or Tags can serve this function for small organizations. The cost of proper software is recouped in staff time saved, improved planning, and potentially reduced taxes.

## Line-by-Line Allocation Guidance

### Compensation (Lines 5-10)

**Line 5: Officers, directors, trustees, key employees**

Allocate based on documented time allocation. Each individual's compensation should be split according to their actual time spent on each function.

| Role | Typical Allocation | Rationale |
|---|---|---|
| Executive Director/CEO | 10-30% program, 50-70% M&G, 10-20% fundraising | Depends on how hands-on in programs vs. admin |
| Program Director | 80-90% program, 10-20% M&G | Primarily program; some admin duties |
| Finance Director/CFO | 0-10% program, 80-90% M&G, 5-10% fundraising | Primarily admin; grant reporting may count as program |
| Development Director | 5-10% program, 5-10% M&G, 80-90% fundraising | Primarily fundraising |
| Board members (if compensated) | Usually 100% M&G | Governance is M&G |

**Lines 6-7: Other salaries and wages**

Allocate by function based on the role. Field staff and program delivery staff are typically 100% program. Administrative support staff may be M&G or split.

**Lines 8-10: Benefits, pension, payroll taxes**

Follow the same allocation percentages as the salary lines they correspond to. If Employee X's salary is 70% program / 20% M&G / 10% fundraising, their benefits and payroll taxes use the same split.

### Professional Fees (Lines 11a-g)

| Line | Category | Typical Allocation |
|---|---|---|
| 11a | Accounting | Usually 100% M&G; but audit of specific program grant may be partly program |
| 11b | Legal | By purpose: employment law = M&G; program-related legal = program |
| 11c | Lobbying | 100% to lobbying (not a standard function; reported separately) |
| 11d | Professional fundraising | 100% fundraising |
| 11e | Investment management | Usually 100% M&G |
| 11g | Other professional | By purpose: program consultants = program; IT consultant = M&G or split |

### Occupancy (Line 16)

Allocate by square footage or headcount. If the entire office is shared:

1. Measure or estimate square footage dedicated to each function
2. Shared spaces (conference room, kitchen, reception): allocate by headcount or proportional use
3. Apply the resulting percentages to all occupancy costs: rent, utilities, property insurance, maintenance

Example: 60% of space is program operations, 30% is admin offices, 10% is development office; allocate rent 60/30/10.

### Travel (Line 17)

Allocate by trip purpose:
- Field visits for program delivery = program
- Conference for professional development = M&G
- Donor meetings = fundraising
- Mixed-purpose trips: allocate by time spent on each activity during the trip

### Depreciation (Line 22)

Allocate by asset use:
- Program equipment (field devices, project vehicles) = program
- Office computers used by admin staff = M&G
- Building depreciation = follow occupancy allocation

### Insurance (Line 23)

- General liability, property insurance: follow occupancy allocation
- D&O insurance: 100% M&G
- Workers' compensation: follow salary allocation
- Professional liability: by function of staff covered

### Other Common Expenses

| Expense | Typical Allocation Approach |
|---|---|
| Office supplies | Headcount by function |
| Postage/shipping | By purpose (program mailings vs. donor mailings vs. admin) |
| Printing | By purpose |
| Telephone/internet | Headcount or usage |
| Conferences/meetings | By purpose of the event |
| Dues and subscriptions | By purpose (program journals = program; accounting software = M&G) |
| Bank fees | Transaction count by function |
| Vehicle expenses | Mileage log by purpose |

## Joint Cost Allocation (Line 26)

### When Joint Cost Allocation Applies

Joint costs arise when a single activity combines elements of two or more functions (most commonly program + fundraising). Example: a direct mail piece that educates the public about an issue (program) AND asks for donations (fundraising).

### ASC 958-720 Three-Test Criteria

Joint cost allocation is ONLY permitted when ALL THREE tests are met:

| Test | Question | How to Evaluate |
|---|---|---|
| **Purpose** | Does the activity accomplish a program or M&G function? Would the activity be conducted even without the fundraising component? | Evidence: program content is substantive, not incidental; activity has measurable program outcomes |
| **Audience** | Is the audience selected based on its need for the program content (not just its likelihood of donating)? | Evidence: audience selection criteria are program-driven, not donor-list driven |
| **Content** | Does the activity include a call to action that would fulfill a program purpose? | Evidence: the content educates, advocates, or serves the audience independent of the ask |

If ANY test fails: 100% of the cost goes to fundraising.

### Allocation Methods for Joint Costs

If all three tests pass, allocate using one of:

1. **Physical units**: proportion of physical space (pages, time) dedicated to each function
2. **Relative direct cost**: ratio of direct costs identifiable to each component
3. **Stand-alone cost**: what each component would cost if performed independently

### Disclosure

If joint costs are allocated (Line 26 > $0):
- Report total joint costs, amount allocated to program, amount allocated to M&G, amount allocated to fundraising
- Describe the allocation method on Schedule O
- Be prepared for IRS scrutiny; joint cost allocation is a common examination trigger

## Common Pitfalls

### Under-Allocating Fundraising
- Grant writing time is fundraising (not program), even though grants fund programs
- Donor cultivation events are fundraising, even if they include program education
- Annual report preparation is partly fundraising if it includes an appeal or is distributed to donors
- Board time spent on fundraising strategy counts as fundraising

### Over-Allocating to Program
- Executive time spent on admin tasks (HR, finance, compliance) is M&G, not program
- Travel to conferences for professional development is M&G, not program
- General organizational planning is M&G

### Inconsistent Allocation Methods
- Changing allocation bases year-over-year without documenting the reason
- Using different bases for similar expense types (e.g., time for salaries but headcount for benefits)
- Not updating allocation percentages when staffing or space usage changes

### Missing Allocations
- Putting 100% of shared costs into M&G instead of allocating across functions
- Not allocating depreciation, insurance, or occupancy
- Not splitting officer/director compensation across functions

## Benchmarks and Reasonableness

While there are no IRS-mandated ratios, these benchmarks help assess reasonableness:

| Metric | Healthy Range | Concern Threshold | Source |
|---|---|---|---|
| Program expense ratio | 65-85% of total expenses | Below 65% may trigger IRS inquiry | IRS examination guidelines |
| M&G ratio | 10-25% of total expenses | Above 25% may draw scrutiny | Industry standard |
| Fundraising ratio | 5-20% of total expenses | Above 25% of related revenue is a red flag | IRS examination guidelines |
| Fundraising efficiency | $0.10-$0.35 per $1.00 raised | Above $0.50 per dollar raised is concerning | Charity watchdog agencies |
| Program + support combined | >= 75% program spending | Sum of M&G + fundraising should not exceed 25% of total | BBB Wise Giving Alliance / National Center for Nonprofit Boards |

The BBB Wise Giving Alliance, regulatory agencies, and funders generally recommend that the sum of management and fundraising costs equal no more than 25% of total expenditures, yielding a minimum 75% program spending ratio. Note: the revised Form 990 no longer displays functional expense totals on the front page (Part I), but the ratios can still be derived from Part IX Line 25 column totals.

These are guidelines, not rules. New organizations, organizations with high earned revenue, and organizations in capital campaigns may have ratios outside these ranges for legitimate reasons. Document the rationale.

## QuickBooks Class Mapping

For organizations using QuickBooks Classes to track functional allocation:

### Recommended Class Structure

| Class Name | Maps to Part IX Column |
|---|---|
| Program: [Program 1 Name] | (B) Program Services |
| Program: [Program 2 Name] | (B) Program Services |
| Program: [Program 3 Name] | (B) Program Services |
| Management & General | (C) Management and General |
| Fundraising | (D) Fundraising |

### Implementation

- Assign a class to every transaction at the point of entry
- For split transactions (e.g., rent shared across functions), use the Split feature to allocate by percentage
- Run P&L by Class report at year-end; this directly feeds Part IX
- For staff who split time across functions, split their payroll charges by class (or allocate at year-end using a journal entry)

If QBO Classes are not set up, the functional allocation must be performed manually (or using a spreadsheet model) during 990 preparation. This is the most common scenario for small organizations and requires careful documentation of the allocation methodology.
