# Construction Terminology Reference

Spanish/Central American construction terms commonly found on plans drawn for Caribbean and Central American projects. Use this reference when interpreting plans that use Spanish-language labels.

## Structural Terms

| Spanish | English | Notes |
|---------|---------|-------|
| Zapata | Pad footing | Isolated footing under a column |
| Cimiento corrido | Continuous strip footing | Runs under walls, connects pad footings |
| Solera de humedad | Moisture barrier beam at grade | Sits at ground level, transitions from foundation to wall |
| Solera intermedia | Intermediate bond beam | Mid-height reinforced concrete beam in walls. Critical for seismic resistance. |
| Solera de corona | Crown bond beam (ring beam) | Top-of-wall beam that ties the wall system at roof level |
| Muro de piedra | Stone masonry wall | Often used between strip footing and grade beam |
| Hormigon ciclopeo | Cyclopean concrete | Large stones embedded in concrete, used for foundation walls |
| Columnas de confinamiento | Tie columns | Reinforced concrete columns that confine the masonry walls (C1, C2, etc.) |
| Castillo | Tie column (Mexican/Central American term) | Same as columna de confinamiento |
| Eslabones / Estribos | Stirrups | U-shaped rebar ties in columns and beams |
| Dentado | Toothing | Staggered block pattern at column locations for mechanical interlock |
| Arranques | Starter bars | Rebar stubs left projecting from foundation for column connection |
| Solado | Blinding layer | Thin concrete layer poured on soil before placing rebar |
| Sobrecimiento | Stem wall | Wall section above foundation, below floor level |
| Viga | Beam | General term |
| Viga canal | Gutter beam / roof gutter | Built-in concrete gutter, common in Caribbean construction |
| Losa | Slab | Floor or roof slab |
| Torta de concreto | Concrete slab (non-structural) | Poured floor topping |
| Tijeras | Trusses | Literally "scissors" -- Central American usage for roof trusses |
| Costanera | C-purlin | C-shaped steel purlin for roof framing |
| Capote | Ridge cap flashing | Metal flashing at roof peak |
| Porta costanera | Purlin bracket | Hardware connecting purlin to beam |
| Cajuela de remate | End cap / flashing trim | Trim piece at roof edges |
| Lamina troquelada | Corrugated sheet metal | Roofing material |
| Flipones | Circuit breakers | Central American slang |
| Poliducto | Flexible conduit (ENT) | Flexible plastic conduit for electrical |
| Tomacorriente | Electrical outlet (receptacle) | Wall outlet |
| Tablero de flipones | Breaker panel (load center) | Main electrical panel |
| Contador electrico | Electric meter | Utility meter |

## Finish Terms

| Spanish | English | Notes |
|---------|---------|-------|
| Repello | Base plaster coat | First coat of the three-coat plaster system |
| Cernido | Finish plaster coat | Second coat, provides smooth surface |
| Pintura | Paint | Third coat, protects against moisture |
| R+C+P | Repello + Cernido + Pintura | Standard three-coat wall finish system |
| Azulejo (AZ) | Ceramic tile | Used on walls in wet areas |
| Piso ceramico (PC) | Ceramic tile floor | Floor tile |
| Cielo falso prefabricado (DFP) | Prefabricated drop ceiling | Suspended ceiling panels |
| Adoquin (AD) | Paving stone | Exterior paving |
| Pulido / Alisado | Polished / Smoothed | Concrete surface treatment |
| Alisado de cemento | Smooth cement finish | Interior render for drain boxes |

## Plumbing and Drainage Terms

| Spanish | English | Notes |
|---------|---------|-------|
| Aguas negras | Sewage (blackwater) | Toilet waste |
| Aguas grises | Greywater | Sink, shower, laundry waste |
| Pluvial | Stormwater (rainwater) | Roof and surface drainage |
| Cajas de registro | Inspection boxes (cleanouts) | Access points for drain maintenance |
| Cajas sifon | Siphon trap boxes (P-trap boxes) | Water-seal chambers that prevent sewer gas |
| Trampa de grasa | Grease trap | Two-chamber grease separator for kitchen drains |
| Bajada de agua pluvial (BAP) | Downpipe / downspout | Roof water discharge pipe |
| Pozo de aguas servidas | Septic pit | Wastewater treatment |
| Campo de absorcion | Absorption field / drain field | Septic effluent dispersal |
| Drenaje pluvial | Stormwater drainage | Surface water management |
| N.P.T. (Nivel de Piso Terminado) | F.F.L. (Finished Floor Level) | Reference elevation for floor |
| Red principal | Public water main | Municipal water supply |
| Ladrillo tayuyo | Solid clay brick | Used for drain box construction |
| Tapadera de concreto | Concrete cover / lid | Removable access cover |

## Window and Door Terms

| Spanish | English | Notes |
|---------|---------|-------|
| Ventana tipo | Window type | Classification label |
| Sillar | Sill height | Bottom of window above floor |
| Dintel | Lintel height | Top of window above floor |
| Puerta tipo | Door type | Classification label |
| Ancho vano | Opening width | Clear width of door/window opening |
| Vidrio bronce | Bronze-tinted glass | Standard solar heat-reducing glazing for tropics |
| Lamina de MDF | MDF panel | Medium-density fiberboard -- degrades quickly in tropical humidity |
| Lamina de zinc | Zinc sheet (galvanized steel) | General-purpose sheet metal |
| Madera de pino tratado | Treated pine wood | Must verify treatment is tropical-grade |

## Measurement and Drawing Conventions

| Term | Meaning |
|------|---------|
| ESCALA 1/100 | Scale 1:100 (1 cm on paper = 1 m in reality) |
| ESCALA 1:20 | Scale 1:20 (detail drawings, 5x larger than 1:100) |
| FASE | Phase (construction phase grouping) |
| PLANO | Plan / drawing sheet |
| PLANTA | Plan view (looking down) |
| SECCION | Section (cut-through view) |
| ELEVACION | Elevation (face-on view) |
| SIMBOLOGIA | Legend / symbol key |
| NOMENCLATURA | Legend / naming key |
| ESPECIFICACIONES | Specifications |
| DETALLE TIPICO | Typical detail (representative for all similar conditions) |
